# Postly - A Mini Backend Project
A simple social posting app built with Node.js, Express, and MongoDB.

## Live Demo
https://postly-6g5f.onrender.com

## Features
- User registration and login with JWT authentication
- Password hashing with bcrypt
- Create and edit posts
- Profile page with all your posts

## Tech Stack
- Node.js / Express
- MongoDB / Mongoose
- EJS (templating)
- JWT & bcrypt

## Run Locally
1. Clone the repo
```bash
git clone https://github.com/shambhaviP910/Postly-.git
cd Postly-
```
2. Install dependencies
```bash
npm install
```
3. Add `.env` file
MONGO_URI=your_mongodb_atlas_uri
JWT_SECRET=your_secret

5. Run the app
```bash
node server.js
```
